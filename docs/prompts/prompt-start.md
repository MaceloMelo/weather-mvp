# Prompt de início — MVP de previsão do tempo

## Papel

Atue como **engenheiro de software sênior**.

## Objetivo

Criar um **MVP de previsão do tempo** usando a API **OpenWeatherMap**, com projeto web em **Vue.js 3**, **Tailwind CSS**, **Pinia** e **Axios**.

## Contexto

- Disponibilizar uma **página** com opção de pesquisar por **cidade**, **estado** ou **país**.
- **Listar** o resultado da API, mostrando como está o tempo no local pesquisado.

## Tarefa

Instalar a **stack indicada** (Vue 3, Tailwind, Pinia, Axios) e preparar o projeto para o MVP.

## Critérios de qualidade

- Sempre criar **microtarefas** para execução.
- Documentar as microtarefas na pasta `docs`, em formato Markdown.
- Garantir **contratos** entre os componentes, preservando qualidade de execução, código e UI.
- Exigir **revisão humana** para decisões importantes.

## Formato de saída

Entregar em **etapas**, com:

- **prioridades**;
- **responsáveis sugeridos**;
- **resultados esperados**;

sempre em **Markdown**.
